Core Capabilities

Built for Forensic Precision

Nine specialized subsystems work in concert to catch what human reviewers miss.

🧠

Ensemble Classifier

Soft-Voting ensemble of Random Forest + Gradient Boosting. Stable predictions across diverse writing styles and generation techniques.

📊

Perplexity Scoring

Measures token-sequence predictability. AI models optimize for high probability, leaving a characteristic low-perplexity fingerprint.

🔤

Stylometric Analysis

POS distributions, syntactic dependency patterns, and function-word ratios reveal the "flat" writing style common in LLM outputs.

📄

PDF Ingestion Pipeline

pdfplumber + PyPDF2 strips multi-column formatting, headers, and metadata — isolating clean body text for reliable NLP processing.

💡

SHAP Explainability

Every prediction comes with SHAP visualization showing which features — vocabulary diversity, syntax repetition — triggered the flag.

🛡️

Adversarial Robustness

Validated against T5-paraphrased content and SCIgen baselines. Minor rephrasing or synonym swaps won't fool the detector.
